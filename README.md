# Smart Garbage Monitoring System ♻

An IoT-based smart city solution for monitoring garbage bins in real time.

## Features

• Real-time garbage bin monitoring  
• Google Maps integration  
• Optimized garbage collection route  
• Citizen complaint system  
• Interactive dashboard

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript
- Google Maps API Note: Google Maps API key is restricted for security reasons.
To run the project:

1. Copy config.example.js
2. Rename to config.js
3. Add your Google Maps API key
Backend:
- Node.js
- Express.js

IoT:
- ESP32
- Ultrasonic Sensor
